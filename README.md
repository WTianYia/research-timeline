# 唐玉超教授研究工作时间轴

一个无前端框架的交互式学术时间轴。页面从 `data/papers.csv` 读取论文数据，以研究方向为泳道，展示论文节点、方法继承关系和自动生成的阶段总结。

## 本地运行

在项目目录执行：

```powershell
python -m http.server 8765
```

浏览器打开 `http://127.0.0.1:8765/`。由于浏览器安全策略限制，不能直接双击 `index.html` 读取 CSV。

## 数据字段

`papers.csv` 包含：`id, year, title, journal, authors, keywords, summary, direction, type, importance, representative, parent_id, doi`。

`data/paper-insights.json` 保存 Grok 4.5 对全部 86 篇文献生成的结构化精读与创新性判断，`data/paper-insights/` 保存逐篇检查点。脚本优先使用 MinerU 全文，无全文时自动退回摘要并降低置信度上限。

重新生成：`node scripts/generate-grok-insights.mjs`

- `direction` 必须对应 `data/directions.json` 中的方向 ID。
- `type` 支持 `theory`、`extension`、`algorithm`、`application`。
- `importance` 建议取 1–5，控制节点大小。
- `representative` 为 `true` 时纳入“仅显示代表作”。
- `parent_id` 可填写一篇或多篇父论文 ID，多项使用 `|` 分隔。
- 泳道按论文主要贡献划分，不能因为包含图像实验就覆盖其算法或理论主线。
- `parent_id` 只表示明确的算法族、模型族或方法继承关系；共同作者或关键词相似不构成连线。
- 完整审计口径与修订记录见 `docs/literature-data-audit-2026-07-13.md`。

## 交互

- 主画布滚轮只进行纵向滚动；年份跨度与泳道高度仅通过右下角缩放按钮调整。
- 首次加载与适配视图会自动压缩到完整展示六条泳道，左侧图例固定在侧栏底部。
- 首次加载默认关闭文献详情，并将 2026 年竖线与右下角浮动工具栏左边缘对齐。
- 论文节点使用扩展的透明命中区，节点边缘仍优先点击；时间轴空白处移动超过 5px 后才进入万向拖拽，轻微抖动不会移动画布。
- 顶部支持关键词搜索和双端年份筛选，左侧支持研究方向、论文类型、创新性及代表作组合筛选。创新性分为“核心创新、显著扩展、增量改进、应用迁移、证据不足”。
- 节点悬停显示简要书目信息，点击后在右侧查看作者、期刊、DOI、摘要、关键词和直接关联论文；重复点击同一节点可关闭详情。
- 节点右上角的 `+N` 表示同一研究方向、同一年另有 N 篇论文；这些论文节点已经全部显示，并非折叠数量。
- 横向移动年份视图时，论文节点与相关关系线会在绘图区边缘同步渐隐，完全离开后同时隐藏。
- 浮动画布工具支持放大、缩小、适配、关系线开关和全屏，底部密度导航可快速定位年份。
- 可将当前筛选和视图导出为 SVG 或 PNG。

## 数据口径

数据由本地核验文献清单生成。方向、论文类型、重要性和继承关系属于研究脉络分析字段，便于可视化，不替代作者本人对研究成果的正式分类。
