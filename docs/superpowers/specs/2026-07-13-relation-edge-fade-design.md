# 关系线边缘同步渐隐设计

## 问题

论文节点在横向拖拽后越过绘图区边缘会停止渲染，但关系线仍使用所有论文的布局坐标，因此连接到已隐藏节点的曲线可能继续横跨可见画布。

## 设计

使用同一个纯函数计算节点在绘图区左右边缘的可见度。节点位于绘图区内部时可见度为 1；进入 20px 边缘缓冲区后线性降至 0；完全离开缓冲区后不再绘制。关系线的可见度取两个端点可见度的较小值，因此任一端点渐隐时整条关系线同步渐隐，端点完全隐藏后关系线停止绘制。

现有普通关系、跨方向虚线、选中高亮和弱化状态继续保留各自基础透明度，再与边缘可见度相乘。

## 验收标准

- 节点与关系线使用相同的左右边缘和 20px 渐隐宽度。
- 任一端点完全不可见时，对应关系线不渲染。
- 拖拽、详情选择、关系线开关和筛选行为不变。
- 自动测试覆盖绘图区内部、渐隐区、边界外和关系线两端取最小值。
- 本机 Chrome 拖动检查不再出现无端点的残留关系线。
