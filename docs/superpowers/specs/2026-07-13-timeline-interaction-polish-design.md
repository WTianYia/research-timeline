# 时间线交互微调规格

## 已确认目标

1. 画布工具栏改为右下角横向排列。
2. 六个研究方向使用内容驱动的泳道高度；论文少的方向明显收紧，论文密集方向保留足够空间。
3. 滚轮围绕指针同时缩放横向年份跨度和纵向泳道高度。
4. 空白区域拖拽同时支持横向时间平移与纵向画布滚动。
5. 首屏默认不显示论文标题卡片；悬停仍显示提示，点击后只显示当前论文标题卡片。
6. 详情面板头部和关闭按钮固定；重复点击同一论文节点关闭详情。

## 交互边界

- 横向缩放范围保持 3–34 年。
- 纵向缩放使用独立倍率，限制为 0.65–1.8，防止泳道过薄或过高。
- 泳道基础高度根据该方向当前结果数量计算，少量论文取紧凑高度，密集方向逐级增加。
- 万向拖拽不修改筛选条件；横向更新年份视窗，纵向更新中央画布自身的 `scrollTop`。
- 默认详情仍显示最新代表作，但不触发节点淡化和标题卡片；用户主动点击后才进入聚焦状态。
- 本轮只进行本地修改与 Chrome 验证，不提交、不推送、不部署。

## 验收标准

- 工具栏位于主画布右下角并横向排列。
- 3 篇和 5 篇方向的泳道高度小于 15–23 篇方向。
- 滚轮一次操作同时改变可见年份跨度和 SVG 总高度。
- 拖拽产生横向年份变化与纵向滚动变化。
- 首屏标题卡片数量为 0；点击一篇后为 1；再次点击同一篇后详情关闭。
- 详情面板滚动时头部及关闭按钮保持可见。

## 2026-07-13 第二轮调整

- 主画布滚轮只执行纵向滚动，不再触发任何横向或纵向缩放；二维缩放仅由右下角放大、缩小和适配按钮触发。
- 首次加载、重置和点击“适配视图”时，根据画布实际可用高度自动计算纵向比例，完整展示六条泳道，并保持节点可辨识、可点击。
- 左侧栏拆分为独立滚动内容区和固定底部图例区；“文献类型”位于滚动区末尾，滚动到底后完整可见，图例始终固定在左下角且不遮挡筛选内容。
