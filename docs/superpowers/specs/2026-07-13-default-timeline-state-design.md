# 默认时间线状态设计

## 目标

桌面端首次打开网站时，保持当前年份跨度和六泳道高度，将 2026 年竖线水平定位到右下角画布工具栏的左边缘，并且不自动打开任何文献详情。

## 设计

- 筛选年份继续保持 2006–2026，不修改数据范围。
- 初始视窗保持 21 年跨度，只调整 `viewStart/viewEnd` 的水平位置。
- 使用主画布实际宽度、SVG 左右边距、工具栏实际宽度和右侧间距，反算使 2026 年落在工具栏左边缘的视窗。
- 初始化和“重置全部”使用该对齐视窗；“适配视图”仍恢复完整年份范围。
- HTML 初始状态直接标记为 `detail-closed`，初始化代码不再自动选中最新代表作。
- 窄屏仍沿用现有响应式布局，不强制执行像素对齐。

## 验收

- 桌面 Chrome 首次加载时，2026 年竖线与工具栏左边缘误差不超过 2px。
- 初始 `selectedId` 为空，详情面板关闭，标题卡数量为 0。
- 86 个节点、50 条关系和六泳道仍完整渲染。
- 点击文献后详情仍能正常打开，再次点击仍能关闭。
